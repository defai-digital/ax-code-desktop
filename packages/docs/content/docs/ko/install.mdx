# 설치

AX Code App를 실행하는 방법은 두 가지입니다:

- macOS용 데스크톱 앱
- CLI가 호스팅하는 local web UI

## 사전 요구 사항

먼저 [AX Code](https://ax-code.ai)를 설치하세요 — AX Code App는 그 위에서 실행됩니다.


## Local web UI

```bash
curl -fsSL https://raw.githubusercontent.com/openchamber/openchamber/main/scripts/install.sh | bash
openchamber --ui-password be-creative-here
```

CLI가 출력한 URL을 여세요(보통 `http://localhost:3000`). AX Code App 세션 목록이 보일 것입니다.

## 데스크톱

GitHub 릴리스 페이지 또는 AX Code App 다운로드 페이지에서 최신 데스크톱 빌드를 내려받으세요. 실행한 뒤 평소 사용하는 AX Code 워크플로로 로그인하세요.
