# 빠른 시작

## 가장 빠른 경로

1. [AX Code](https://ax-code.ai)를 설치합니다.
2. AX Code App CLI를 설치합니다(한 줄 명령은 [설치](/ko/install/)를 참고하세요).
3. `openchamber --ui-password be-creative-here`를 실행합니다.
4. CLI가 출력한 URL을 엽니다(보통 `http://localhost:3000`).
5. 신뢰할 수 있는 데스크톱 또는 워크스테이션 브라우저에서 세션을 유지하세요.

브라우저에 AX Code App 세션 목록이 보일 것입니다. 화면이 뜨면 정상적으로 실행된 것입니다.

특히 터널이나 reverse proxy로 인스턴스를 공개할 계획이라면 강력한 UI 비밀번호를 사용하세요.

페이지가 열리지 않으면 [문제 해결](/ko/troubleshooting/)을 확인하세요.

## 어떤 앱을 사용해야 하나요?

- macOS에서의 일상 작업에는 **데스크톱**을 사용하세요
- development와 제어된 워크스테이션 접근에는 **local web**을 사용하세요
