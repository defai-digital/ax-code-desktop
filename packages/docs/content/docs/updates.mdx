# Updates

How you update AX Code Desktop depends on how you installed it. In every case you can check your current version at **Settings → AX Code Desktop → About**.

## Desktop app

The desktop app checks for updates against GitHub releases. When one is available, AX Code Desktop tells you, downloads it when you choose, and installs it on the next restart. You're always in control — nothing installs without your say-so.

## Web / CLI

If you installed the CLI, update it from **About** with the check and update buttons, or from the terminal:

```bash
ax-code-desktop update
```

AX Code Desktop detects how it was installed (npm, pnpm, yarn, or bun) and runs the right update for you.

## The AX Code server

AX Code Desktop and AX Code update separately. When a new AX Code version is available, AX Code Desktop offers to update it and restarts the server afterward. See [AX Code Server](/ax-code-server/) for how that server is managed.
