# Швидкий старт

## Найшвидший шлях

1. Встановіть [AX Code](https://ax-code.ai).
2. Встановіть CLI AX Code App (команду в один рядок дивіться в [Встановленні](/uk/install/)).
3. Виконайте `openchamber --ui-password be-creative-here`.
4. Відкрийте URL, який вивів CLI (зазвичай `http://localhost:3000`).
5. Тримайте сесію на довіреному десктопі або в браузері робочої станції.

У браузері ви маєте побачити список сесій AX Code App. Якщо він завантажився — все працює.

Використовуйте надійний пароль UI, особливо якщо плануєте відкривати інстанс через тунель або reverse proxy.

Якщо сторінка не завантажується, перегляньте [Усунення несправностей](/uk/troubleshooting/).

## Яку платформу обрати?

- обирайте **десктоп** для щоденної роботи на macOS
- обирайте **local web** для розробки та контрольованого доступу з робочої станції
