# Оновлення

Те, як ви оновлюєте AX Code App, залежить від того, як ви його встановили. У будь-якому разі ви можете перевірити свою поточну версію в **Settings → AX Code App → About**.

## Десктопний застосунок

Десктопний застосунок перевіряє наявність оновлень у релізах GitHub. Коли воно доступне, AX Code App повідомляє вам, завантажує його, коли ви оберете, і встановлює при наступному перезапуску. Ви завжди керуєте процесом — нічого не встановлюється без вашого схвалення.

## Веб / CLI

Якщо ви встановили CLI, оновіть його з **About** кнопками перевірки та оновлення або з терміналу:

```bash
openchamber update
```

AX Code App визначає, як його було встановлено (npm, pnpm, yarn чи bun), і запускає правильне оновлення за вас.

## Сервер AX Code

AX Code App та AX Code оновлюються окремо. Коли доступна нова версія AX Code, AX Code App пропонує оновити її й перезапускає сервер після цього. Див. [AX Code Server](/uk/ax-code-server/) про те, як цим сервером керують.
