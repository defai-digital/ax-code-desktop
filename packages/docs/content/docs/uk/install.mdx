# Встановлення

There are two supported ways to run AX Code App:

- десктоп-застосунок для macOS
- local web UI hosted by the CLI for development and controlled workstation access

## Передумова

Спершу встановіть [AX Code](https://ax-code.ai) — AX Code App працює поверх нього.


## Local web UI

```bash
curl -fsSL https://raw.githubusercontent.com/openchamber/openchamber/main/scripts/install.sh | bash
openchamber --ui-password be-creative-here
```

Відкрийте URL, який вивів CLI (зазвичай `http://localhost:3000`). Ви маєте побачити список сесій AX Code App.

## Десктоп

Завантажте найновішу десктопну збірку зі сторінки релізів на GitHub або зі сторінки завантаження AX Code App. Відкрийте її та увійдіть у свій звичний робочий процес AX Code.
