# Instancias remotas

La app de escritorio puede conectarse a AX Code App ejecutándose en otra máquina por SSH —un servidor de trabajo, una máquina en la nube, un homelab— y traer su UI a tu pantalla como si fuera local. Configúralo en **Settings → Remote Instances**.

> Las instancias remotas son una función **solo de escritorio**. En la web, conéctate a un servidor remoto con las variables de entorno de [AX Code Server](/es/ax-code-server/) en su lugar.

## Añade una instancia remota

1. Abre **Settings → Remote Instances** y añade una.
2. Dale el comando SSH que usarías normalmente para alcanzar la máquina, más un apodo.
3. Elige cómo se ejecuta AX Code App allí:
   - **managed** — AX Code App se instala y se inicia por sí mismo en la máquina remota
   - **external** — conéctate a una que ya está en funcionamiento
4. Conecta.

AX Code App recorre los pasos —comprobar la conexión, configurar el remoto, iniciar el servidor y reenviar el puerto— y muestra en qué punto está en cada etapa. Cuando llega a **ready**, la UI remota se carga localmente.

## Credenciales

Tú decides si guardar las contraseñas de SSH y de UI o introducirlas cada vez. Si la conexión se cae, AX Code App informa qué paso falló para que puedas arreglarlo; consulta [Acceso remoto](/es/troubleshooting/remote-access/).
