# Inicio rápido

## La vía más rápida

1. Instala [AX Code](https://ax-code.ai).
2. Instala la CLI de AX Code App (consulta [Instalación](/es/install/) para el comando de una sola línea).
3. Ejecuta `openchamber --ui-password be-creative-here`.
4. Abre la URL que imprime la CLI (normalmente `http://localhost:3000`).
5. Mantén la sesión en un escritorio o navegador de estación de trabajo de confianza.

Deberías ver la lista de sesiones de AX Code App en tu navegador. Si carga, ya está todo en marcha.

Usa una contraseña de UI fuerte, sobre todo si piensas exponer la instancia mediante un túnel o proxy inverso.

Si la página no carga, consulta [Resolución de problemas](/es/troubleshooting/).

## ¿Qué app debería usar?

- usa **escritorio** para el día a día en macOS
- use **local web** for development and controlled workstation access
