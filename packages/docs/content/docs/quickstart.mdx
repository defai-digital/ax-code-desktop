# Quickstart

## Fastest path

1. Install [AX Code](https://ax-code.ai).
2. Install the AX Code Desktop CLI or use a development checkout (see [Install](/install/)).
3. Run `ax-code-desktop --ui-password be-creative-here`.
4. Open the URL the CLI prints (usually `http://localhost:3000`).
5. Keep the session on a trusted desktop or workstation browser.

You should see the AX Code Desktop session list in your browser. If it loads, you're up and running.

Use a strong UI password, especially if you expose the instance through a reverse proxy or bind it outside localhost.

If the page doesn't load, check [Troubleshooting](/troubleshooting/).

## Which app should I use?

- use **desktop** for day-to-day work on macOS
- use **local web** for development and controlled workstation access
