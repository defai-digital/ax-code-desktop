# Install

There are three ways to run AX Code App:

- desktop app for macOS
- web app hosted by the CLI, which you can install like a phone app (a PWA)
- VS Code extension

## Prerequisite

Install [AX Code](https://ax-code.ai) first — AX Code App runs on top of it.

## Web + PWA

```bash
curl -fsSL https://raw.githubusercontent.com/openchamber/openchamber/main/scripts/install.sh | bash
openchamber --ui-password be-creative-here
```

Open the URL the CLI prints (usually `http://localhost:3000`). You should see the AX Code App session list. To keep it handy, use your browser's "Install" option in the address bar to add it as an app.

## Desktop

Download the latest desktop build from the GitHub releases page or the AX Code App download page. Open it and sign into your usual AX Code workflow.

## VS Code

Install from the VS Code Marketplace and sign into your usual AX Code workflow. The AX Code App view then opens in the sidebar.
