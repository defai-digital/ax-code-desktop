# Szybki start

## Najszybsza droga

1. Zainstaluj [AX Code](https://ax-code.ai).
2. Zainstaluj CLI AX Code App (jednowierszowe polecenie znajdziesz w sekcji [Instalacja](/pl/install/)).
3. Uruchom `openchamber --ui-password be-creative-here`.
4. Otwórz adres URL wypisany przez CLI (zwykle `http://localhost:3000`).
5. Utrzymuj sesję na zaufanym desktopie albo w przeglądarce stacji roboczej.

W przeglądarce powinna pojawić się lista sesji AX Code App. Jeśli się załadowała, wszystko działa.

Używaj silnego hasła UI, zwłaszcza jeśli planujesz udostępnić instancję w internecie.

Jeśli strona się nie ładuje, zajrzyj do [Rozwiązywania problemów](/pl/troubleshooting/).

## Której platformy użyć?

- użyj **komputera** do codziennej pracy na macOS
- użyj **lokalnej wersji web** do developmentu i kontrolowanego dostępu ze stacji roboczej
