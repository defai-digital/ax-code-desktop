# Projekty

Projekt to folder na Twoim komputerze, który AX Code App śledzi — zwykle jedna baza kodu. Przełączenie projektu zmienia folder, w którym pracuje agent, wraz z sesjami i ustawieniami tego projektu.

## Dodaj projekt

Projekt możesz dodać z kilku miejsc:

- pozycja **Add project** w palecie poleceń
- przycisk **+** u góry paska bocznego sesji
- przeglądarka folderów, gdy wybierasz katalog

Wskaż folder, a AX Code App go zapamięta. Nazwa pochodzi od folderu; możesz ją później zmienić.

## Przełączaj projekty

Wybierz projekt z paska bocznego, aby go aktywować. Wszystko — sesje, git, notatki — podąża za otwartym projektem.

## Spraw, by projekt był rozpoznawalny

Otwórz **Settings → Projects**, aby nadać projektowi własną nazwę, kolor lub ikonę. AX Code App próbuje znaleźć ikonę automatycznie — zobacz [Ikony projektów](/pl/project-icons/).
