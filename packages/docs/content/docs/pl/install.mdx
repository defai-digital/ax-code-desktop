# Instalacja

AX Code App można uruchomić na dwa sposoby:

- aplikacja na komputer dla macOS
- lokalny web UI hostowany przez CLI

## Wymaganie wstępne

Najpierw zainstaluj [AX Code](https://ax-code.ai) — AX Code App działa na jego bazie.


## Local web UI

```bash
curl -fsSL https://raw.githubusercontent.com/openchamber/openchamber/main/scripts/install.sh | bash
openchamber --ui-password be-creative-here
```

Otwórz adres URL wypisany przez CLI (zwykle `http://localhost:3000`). Powinna pojawić się lista sesji AX Code App.

## Komputer

Pobierz najnowszą kompilację na komputer ze strony wydań na GitHubie lub ze strony pobierania AX Code App. Otwórz ją i zaloguj się do swojego zwykłego procesu pracy AX Code.
