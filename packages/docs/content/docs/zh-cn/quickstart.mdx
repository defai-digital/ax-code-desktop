# 快速开始

## 最快路径

1. 安装 [AX Code](https://ax-code.ai)。
2. 安装 AX Code App CLI（一行命令见[安装](/zh-cn/install/)）。
3. 运行 `openchamber --ui-password be-creative-here`。
4. 打开 CLI 输出的 URL（通常是 `http://localhost:3000`）。
5. 只在可信桌面或工作站浏览器中保留会话。

你应该会在浏览器中看到 AX Code App 会话列表。如果它能加载出来，说明你已经成功运行起来了。

请使用强 UI 密码，尤其是当你打算通过隧道或反向代理开放实例时。

如果页面无法加载，请查看[问题排查](/zh-cn/troubleshooting/)。

## 我应该使用哪个应用？

- 在以 macOS 为主的日常工作中使用 **桌面**
- 在开发和受控工作站访问中使用 **local web**
