# 安装

There are two supported ways to run AX Code App:

- 面向 macOS 的桌面应用
- local web UI hosted by the CLI for development and controlled workstation access

## 前提条件

请先安装 [AX Code](https://ax-code.ai)——AX Code App 在它之上运行。


## Local web UI

```bash
curl -fsSL https://raw.githubusercontent.com/openchamber/openchamber/main/scripts/install.sh | bash
openchamber --ui-password be-creative-here
```

打开 CLI 输出的 URL（通常是 `http://localhost:3000`）。你应该会看到 AX Code App 会话列表。

## 桌面

从 GitHub 发布页或 AX Code App 下载页下载最新的桌面版本。打开它，并登录你常用的 AX Code 工作流。
