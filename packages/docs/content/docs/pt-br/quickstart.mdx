# Início rápido

## Caminho mais rápido

1. Instale o [AX Code](https://ax-code.ai).
2. Instale a CLI do AX Code App (veja [Instalação](/pt-br/install/) para o comando de uma linha).
3. Execute `openchamber --ui-password be-creative-here`.
4. Abra a URL que a CLI imprime (geralmente `http://localhost:3000`).
5. Mantenha a sessão em um desktop ou navegador de estação de trabalho confiável.

Você deve ver a lista de sessões do AX Code App no navegador. Se ela carregar, está tudo funcionando.

Use uma senha de UI forte, principalmente se pretende expor a instância por túnel ou proxy reverso.

Se a página não carregar, confira a [Solução de problemas](/pt-br/troubleshooting/).

## Qual plataforma devo usar?

- use **desktop** para o uso diário no macOS
- use **web local** para desenvolvimento e acesso controlado por estação de trabalho
