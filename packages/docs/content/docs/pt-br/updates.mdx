# Atualizações

Como você atualiza o AX Code App depende de como o instalou. Em todos os casos, você pode verificar sua versão atual em **Settings → AX Code App → About**.

## App de desktop

O app de desktop verifica atualizações nas releases do GitHub. Quando há uma disponível, o AX Code App avisa você, baixa-a quando você escolhe e a instala no próximo reinício. Você está sempre no controle — nada é instalado sem a sua autorização.

## Web / CLI

Se você instalou a CLI, atualize-a a partir de **About** com os botões de verificar e atualizar, ou pelo terminal:

```bash
openchamber update
```

O AX Code App detecta como foi instalado (npm, pnpm, yarn ou bun) e executa a atualização certa para você.

## O servidor AX Code

O AX Code App e o AX Code são atualizados separadamente. Quando uma nova versão do AX Code está disponível, o AX Code App oferece atualizá-la e reinicia o servidor em seguida. Veja [AX Code Server](/pt-br/ax-code-server/) para como esse servidor é gerenciado.
