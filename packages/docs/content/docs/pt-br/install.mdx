# Instalação

Há três formas de rodar o AX Code App:

- app de desktop para macOS
- app web hospedado pela CLI, que você pode instalar como um app de celular (um PWA)
- extensão para VS Code

## Pré-requisito

Instale o [AX Code](https://ax-code.ai) primeiro — o AX Code App roda em cima dele.

## Web + PWA

```bash
curl -fsSL https://raw.githubusercontent.com/openchamber/openchamber/main/scripts/install.sh | bash
openchamber --ui-password be-creative-here
```

Abra a URL que a CLI imprime (geralmente `http://localhost:3000`). Você deve ver a lista de sessões do AX Code App. Para deixá-lo à mão, use a opção "Instalar" do seu navegador na barra de endereço para adicioná-lo como um app.

## Desktop

Baixe a build de desktop mais recente na página de releases do GitHub ou na página de download do AX Code App. Abra-a e entre no seu fluxo de trabalho habitual do AX Code.

## VS Code

Instale pela VS Code Marketplace e entre no seu fluxo de trabalho habitual do AX Code. A visualização do AX Code App então abre na barra lateral.
