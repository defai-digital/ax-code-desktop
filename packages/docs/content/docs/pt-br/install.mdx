# Instalação

Há duas formas compatíveis de rodar o AX Code App:

- app de desktop para macOS
- interface web local hospedada pela CLI

## Pré-requisito

Instale o [AX Code](https://ax-code.ai) primeiro — o AX Code App roda em cima dele.


## Local web UI

```bash
curl -fsSL https://raw.githubusercontent.com/openchamber/openchamber/main/scripts/install.sh | bash
openchamber --ui-password be-creative-here
```

Abra a URL que a CLI imprime (geralmente `http://localhost:3000`). Você deve ver a lista de sessões do AX Code App.

## Desktop

Baixe a build de desktop mais recente na página de releases do GitHub ou na página de download do AX Code App. Abra-a e entre no seu fluxo de trabalho habitual do AX Code.
