# Instâncias Remotas

O app de desktop pode conectar ao AX Code App rodando em outra máquina via SSH — um servidor de trabalho, uma máquina na nuvem, um homelab — e trazer a UI dela para a sua tela como se fosse local. Configure isso em **Settings → Remote Instances**.

> As instâncias remotas são um recurso **apenas para desktop**. Na web, conecte a um servidor remoto com as variáveis de ambiente em [AX Code Server](/pt-br/ax-code-server/) em vez disso.

## Adicionar uma instância remota

1. Abra **Settings → Remote Instances** e adicione uma.
2. Forneça o comando SSH que você normalmente usaria para alcançar a máquina, mais um apelido.
3. Escolha como o AX Code App roda ali:
   - **managed** — o AX Code App se instala e inicia na máquina remota
   - **external** — conecte a um que já está em execução
4. Conecte.

O AX Code App percorre as etapas — verificando a conexão, configurando o remoto, iniciando o servidor e encaminhando a porta — e mostra em que ponto está em cada estágio. Quando chega a **ready**, a UI remota carrega localmente.

## Credenciais

Você decide se quer salvar as senhas SSH e de UI ou inseri-las a cada vez. Se a conexão cair, o AX Code App informa qual etapa falhou para você corrigi-la — veja [Acesso remoto](/pt-br/troubleshooting/remote-access/).
